  • "BI Library" - E-Commerce Website using ReactJS

    Ultraverse - NFT E-Commerce Page (FES Internship)

    HTML, CSS, JavaScript, React

    Fetched multiple data APIs to make each aspect of this website not only dynamic, but responsive and consistent with all other pages of the website. Used a sorting API to create the "Explore" page, where the NFTs can be sorted by price or most liked. Also became familiar with AOS animations!

    Search Engine

    HTML, CSS, JavaScript, React, Firebase

    Fetched a Google Search API to create a fully functional search engine resembling the worldwide Google. For this project, I used React's state and action exports to obtain any search term that is typed into the searchbar.

    Flixer - A Movie Search API Database

    HTML, CSS, JavaScript, React

    Using the omdbAPI database, I created a fully dynamic and responsive movie search engine. It is equipped with an info page for any movie searched for as well as a skeleton loading state upon searching for a set of movies. In this project, I utilized the useState useParams properties in react in order to fetch the API and portray the correct search results.

    Movie Sorting Homepage (Hulu)

    HTML, CSS, JavaScript, React, NextJS, Tailwind CSS, Vercel

    Created a constantly updataing movie homepage that sorts over 2,500 streaming products by category. Using NextJS and TailwindCSS, I was able to create a fully responsive lazy-loading page that automatically shows more movies as the user scrolls down. This website is not only compatible for small screens and tablets, but is also compatible for any screen larger than a regular 24" computer monitor.
